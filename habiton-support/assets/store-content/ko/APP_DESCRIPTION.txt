# HabitOn - 앱 스토어 설명 (KO)

## 전체 설명 (iOS & Android)

HabitOn으로 하루하루 습관을 쌓아보세요. 단순하지만 강력한 기능으로 지속 가능한 변화를 도와드립니다.

🎯 더 나은 습관 만들기
매일/주중/주말/매주/격주/매월, 혹은 사용자 지정 주기로 원하는 습관을 만들고 추적하세요. 라이프스타일에 맞게 유연하게 설정할 수 있습니다.

✨ 유연한 스케줄링
• 매일 규칙적인 습관
• 평일 위주의 업무 습관
• 원하는 요일의 주간 루틴
• 격주 반복으로 꾸준함 유지
• 매월 반복으로 장기 목표 달성
• 완전 사용자 정의 반복 주기

📊 진행 상황 추적
현재/최장 연속 기록과 완료율을 자동으로 계산하여 동기부여를 제공합니다. 쌓여가는 기록으로 성장을 확인하세요.

🔔 스마트 알림
여러 습관이 같은 시간이라면 하나의 알림으로 깔끔하게 병합합니다. 모든 알림은 로컬에서 동작하며 계정 없이 사용할 수 있습니다.

📈 인사이트 제공
• 시각적인 완료 캘린더
• 연속 기록(스트릭)과 마일스톤
• 완료율 통계
• 과거 데이터 확인
• 카테고리별 정리

🎨 개인화 경험
건강/생산성/학습 등 미리 준비된 카테고리를 사용하거나, 아이콘과 색상을 지정해 나만의 습관을 만들 수 있습니다.

🔒 내 데이터, 내 기기
모든 데이터는 기기에만 저장됩니다. 계정 가입이나 서버 전송이 없어 개인정보 걱정이 없습니다.

💪 성공을 위한 설계
• 직관적인 인터페이스
• 날짜 이동으로 놓친 날 기록
• 유연한 습관 관리
• 광고 없음, 방해 없음
• 오프라인에서도 작동

📱 이런 분께 딱 맞아요
• 아침 루틴 만들기
• 운동/건강 목표 추적
• 새로운 기술 학습
• 나쁜 습관 끊기
• 꾸준함 유지
• 자기계발/생산성 향상

🌟 왜 HabitOn인가요?
중요한 것에만 집중합니다. 복잡한 설정 없이 매일 실천하도록 돕는 간단하고 효과적인 습관 관리 앱입니다.

지금 HabitOn과 함께 더 나은 일상을 시작하세요.
