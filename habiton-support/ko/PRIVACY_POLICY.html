HabitOn 개인정보 처리방침

최종 수정일: 2025년 10월 13일

간단 요약: HabitOn은 사용자의 개인 데이터를 수집, 저장, 또는 공유하지 않습니다. 모든 정보는 오직 사용자의 기기 내에만 저장됩니다.

1. 정보 수집 및 이용

HabitOn은 사용자의 프라이버시를 최우선으로 고려하여 설계되었습니다. 본 앱은 개인 정보를 외부 서버에 수집, 전송, 또는 저장하지 않습니다. 앱 내에서 생성되는 모든 데이터(습관, 완료 기록, 설정, 환경설정 등)는 기기의 안전한 저장 시스템을 통해 로컬에 저장됩니다.

2. 데이터 저장

모든 앱 데이터는 기기 내에 로컬로 저장됩니다 (iOS의 경우 AsyncStorage, Android의 경우 SharedPreferences 사용). 저장되는 데이터에는 다음이 포함됩니다:

  • 습관 이름, 설명, 일정
  • 완료 기록 및 연속 달성 데이터
  • 알림 설정
  • 앱 환경설정

이 데이터는 절대로 기기를 벗어나지 않습니다. 저희는 해당 데이터에 접근할 수 없으며, 어떠한 서버나 제3자에게도 전송되지 않습니다.

3. 계정 불필요

HabitOn은 계정 생성, 이메일 입력, 또는 개인정보 제공을 요구하지 않습니다. 완전히 익명으로 앱을 사용할 수 있습니다.

4. 제3자 서비스 미사용

HabitOn은 어떠한 제3자 분석, 광고, 또는 추적 서비스와도 연동되지 않습니다. 다음과 같은 서비스는 전혀 사용하지 않습니다:

  • 분석 도구 (Google Analytics, Firebase Analytics 등)
  • 광고 네트워크
  • 소셜 미디어 연동
  • 데이터를 전송하는 크래시 리포팅 서비스

5. 권한 요청

HabitOn은 다음과 같은 권한만 요청합니다:

  • 알림: 사용자가 설정한 시간에 습관 리마인더를 보내기 위한 용도로만 사용됩니다. 모든 알림은 로컬 알림이며, 외부로 데이터가 전송되지 않습니다.
  • 저장소: 습관 데이터를 기기 내에 저장하기 위해 사용됩니다.

이 외의 권한은 요청하거나 사용하지 않습니다.

6. 데이터 보안

모든 데이터는 사용자의 기기 내에 저장되므로, 기기의 보안 설정(비밀번호, 생체인증, 암호화 등)에 의해 보호됩니다. 다음을 권장합니다:

  • 기기에 비밀번호 또는 생체인증을 설정하기
  • 운영체제를 최신 상태로 유지하기
  • 기기에 물리적으로 접근할 수 있는 사람을 주의하기

7. 데이터 삭제

사용자는 자신의 데이터에 대해 완전한 통제권을 가집니다:

  • 앱 내에서 개별 습관을 언제든 삭제 가능
  • 프로필 설정의 “모든 데이터 초기화” 옵션을 통해 전체 데이터 삭제 가능
  • 앱 삭제 시 해당 앱과 관련된 모든 데이터가 기기에서 제거됩니다

8. 데이터 백업 없음

HabitOn은 클라우드 백업이나 동기화 기능을 제공하지 않습니다. 데이터는 오직 사용자의 기기에만 존재합니다. 앱을 삭제하거나 기기를 분실할 경우 데이터는 복구되지 않습니다. 다음을 권장합니다:

  • 기기의 백업 기능(iCloud, Google 백업 등)을 이용해 앱 데이터를 포함한 백업 생성
  • 기기를 안전하게 관리하기

9. 아동의 개인정보

HabitOn은 13세 미만을 포함한 어떠한 사용자로부터도 정보를 수집하지 않습니다. 데이터 수집이나 전송이 없으므로 모든 연령층이 안전하게 사용할 수 있습니다. 다만 어린이의 경우 보호자의 지도를 권장합니다.

10. 개인정보 처리방침 변경

본 개인정보 처리방침은 필요에 따라 변경될 수 있습니다. 변경 시 본 문서 상단의 “최종 수정일”을 업데이트하여 반영합니다. 변경 이후에도 앱을 계속 사용할 경우, 변경된 방침에 동의한 것으로 간주됩니다.

12. 이용자의 권리

HabitOn은 어떠한 개인 데이터도 수집하거나 처리하지 않으므로:

  • 당사 서버에서 접근, 수정, 또는 삭제할 데이터가 없습니다
  • 이전하거나 이동할 데이터가 없습니다
  • 자동화된 의사결정 또는 프로파일링이 없습니다

모든 데이터는 항상 사용자의 기기 내에서 직접적으로 통제됩니다.

결론: HabitOn은 완전한 프라이버시를 위해 설계되었습니다. 앱은 서버와 통신하지 않기 때문에 데이터를 수집할 수도 없습니다. 사용자의 습관, 진행 상황, 그리고 데이터는 전부 기기 내에만 머뭅니다.
